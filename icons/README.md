# Icons

This directory should contain the extension icons in PNG format:

- icon16.png - 16x16 pixels (toolbar icon)
- icon48.png - 48x48 pixels (extension management page)
- icon128.png - 128x128 pixels (Chrome Web Store)

## Quick Icon Generation

You can generate simple icons using online tools like:
- https://favicon.io/favicon-generator/
- https://www.canva.com/
- Or use the provided SVG and convert to PNG

## Icon Design

Suggested design: A magnifying glass with a checkmark, or the letter "S" with a spellcheck indicator.

Colors: Use #667eea (primary purple) as the main color with white elements.
